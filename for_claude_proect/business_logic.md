# Бизнес-логика проекта

## Монетизация
1. Продажа цифровых книг (основной доход)
2. Премиум подписки (расширенный доступ)
3. Индивидуальные терапевтические сказки (планируется)
4. Пожертвования (дополнительно)

## Пользовательские роли
- Гость: просмотр бесплатного контента
- Зарегистрированный: покупки, избранное, комментарии
- Подписчик: премиум контент, ранний доступ
- Админ: управление контентом

## Терапевтические сказки - Уникальная функция
- Фильтрация по возрасту: 3-5, 6-8, 9-12 лет
- Категории проблем: страхи, самооценка, отношения, поведение
- Цветовое кодирование для быстрой навигации
- Рекомендации на основе выбранных проблем
- Интеграция с аудио-версиями

## Безопасность семейного контента
- Модерация всего пользовательского контента
- Родительский контроль для детских разделов
- GDPR compliance для детских данных
- Защита от неподходящего контента
